## ⚙ 설치

```
# clone the project
$ git clone https://github.com/Ubermensch0608/request_form_rewind.git

# install modules
$ cd requests-for-partner
$ yarn install

# build
$ yarn build

# start
$ yarn start:dev

⠀
⠀  You can now view this project in the browser.
⠀  http://localhost:3001/
```
